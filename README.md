# Jekyll Modern Blog

## Image Standards
Use always images with 1920x1200 pixels.
Image name in lowercase and equal to article.
The preference format: JPG, PNG or TIFF.

## Text Standards
The maximum size for texts is 1000 characters or 150 words.
